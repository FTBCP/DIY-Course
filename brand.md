# Brand Guide — DIY Courses

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| Warm White | `#F5F1E8` | Page background |
| Dark Brown | `#1A1614` | Primary text, dark UI (sidebar, CTA buttons) |
| Sidebar Hover | `#2A2420` | Sidebar hover/active states |
| Terracotta | `#C4553F` | Accent — links, active states, hover on CTA |
| Warm Gray | `#5C4A3A` | Secondary text |
| Muted Tan | `#8B6F4E` | Eyebrow text, labels, footnotes |
| Light Tan | `#A89680` | Placeholders, disabled states |
| Border | `#E0D5C0` | Input borders, dividers |
| Input Background | `#FAF6EC` | Form inputs, source blocks |
| Green Complete | `#5C7A3A` | Completed lesson checkmarks |
| Green Pill BG | `#1F3A1F` | Completed pill background |
| Green Pill Text | `#D8E8C4` | Completed pill text |

## Typography

| Role | Font | Weight | Notes |
|------|------|--------|-------|
| Headings, lesson body | Fraunces | 400–700 | Serif, optical sizing auto |
| UI text, labels, buttons | Inter | 400–700 | Sans-serif |

Google Fonts import:
```
https://fonts.googleapis.com/css2?family=Fraunces:ital,opsz,wght@0,9..144,300;0,9..144,400;0,9..144,500;0,9..144,600;0,9..144,700;1,9..144,400&family=Inter:wght@400;500;600;700&display=swap
```

## Shape

- Inputs and cards: `border-radius: 4px`
- Chips and pills: `border-radius: 999px` (fully rounded)
- Buttons: `border-radius: 4px`

## Tone

- Warm, literate, unhurried
- Serif headings give a "bookish" quality
- Sans-serif UI keeps things clean and functional
- No harsh colors, no neon, no stark white backgrounds
